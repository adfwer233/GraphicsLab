# Analytic Surfaces

GraphicsLab includes closed-form analytic surfaces that are efficient for evaluation and differential queries.

## Implemented Families

- Plane
- Sphere
- Cone
- Cylinder
- Torus

## Why Analytic Surfaces

- Stable derivatives for optimization and projection
- Faster evaluation than generic spline sampling
- Useful as baselines for intersector validation
